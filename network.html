Chapter 4 Terms
Term Definition
Network Classifications
  • LAN
  • MAN
  • WAN
  • Local Area Network (LAN) normally consists of a collection of computers in a single building complex
  • Metropolitan Area Network (MAN) is a network of intermediate size, such as one spanning a local community
  • Wide Area Network (WAN) links machines over a greater distance, perhaps in neighboring cities or on opposite sides of the world
Network Models
  • Bus Network
  • Star Network
  • The machines are all conected to a common communication line
  • One machine serves as a central focal point to which all the others are connected
Protocols
  • CSMA/CD Ethernet
  • CSMA/CA Wifi
  • Carrier Sense, Multiple Access with Collision Detection (CSMA/CD) is a medium access control method used most notably in early Ethernet technology for local area networking
  • Carrier Sense, Multiple Access with Collision Avoidance (CSMA/CA) is a protocol for carrier transmission in 802.11 networks
Combining Networks
  • Repeater
  • Bridge
  • Switch
  • Router
  • Gateway
  • Is little more than a device that passes signals back and forth between the two original buses (usually with some form of amplification) without considering the meaning of the signals
  • Connects two buses, but it does not necessarily pass all messages acress the connection
  • Produces a network consisting of several buses extending from the switch, as spokes on a wheel
  • Special purpose computers used for forwarding messages
  • Serves as a passageway between the network and the outside world
Client/Server Model
  • P2P: peer-to-peer
  • Distributed System
  • Cloud Computing
  • Involves processes that provide service to and recieve service from each other
  • Consists of software units that execute as processes on different computers
  • Huge pools of shared computers on the network can be allocated for use by clients as needed
Internet Addressing
  • Tier 1
  • Tier 2
  • tier 3
  • Consists of very high-speed, high-capacity, international WANs
  • Tend to be more regional in scope and less potent in their capabilities
  • An independent internet, sometimes called an intranet, operated by a single authority that is in the business of supplying Internet access to individual homes and businesses
IP Address
  • ICANN
  • DNS
  • TLD: Top-level Domain
  • Country-code TLDs
  • Internet Corporation for Assigned Names and Numbers (ICANN) is a nonprofit corporation established to coordinate the Internet's operation
  • Domain Name System (DNS) is name servers used aas an Internet-wide directory syste
  • Top-level Domains (TLDs) is one of the domains at the highest level in the hierarchical DNS of the Internet after the root domain
  • An Internet top-level domain generally used or reserved for a country, soveraign state, or dependent territory identified with a country code
Internet Protocols
  • FTP
  • HTTP
  • SMTP
  • File Transfer Protocol (FTP) is a network protocol for transmitting files between computers over Transmission Control Protocol/Internet Protocol (TCP/IP) connections
  • Hypertext Transfer Protocol (HTTP) ia an application layer protocol in the Internet protocol suite model for distributed, collaborative, hypermedia information systems
  • Simple Mail Transfer Protocol (SMTP) is an Internet standard communication protocol for electronic mail transmission
World Wide Web
  • URL
  • Client-side Activities
  • Server-side Activities
  • Uniform Resource Locator (URL) is the address of a web page
  • Activities that are performed by a client such as a browser
  • Activities that are performed by a server such as a webserver
Internet Four-layer Software
  • Application Layer
  • Transport Layer
  • Network Layer
  • Link Layer
  • Specifies the shared communication protocols and interface methods used by hosts in a communications network
  • Provides end-to-end communication services for applications
  • Responsible for packet forwarding including routing through intermediate routers
  • Is the group of methods and communications protocols confined to the link that a shost is physically connected to
Internet is:
  • Packet-switching Network
  • Fault Tolerant
  • Redundant
  • A kind of computer communications network that sends data in the form of small packets
  • The resilient property that enables a system to continue operating properly in the event of failure or major dysfunction in one or more of its components
  • System design in which a component is duplicated so if it fails there will be a backup
Cybersecurity
  • Malware
  • Virus
  • Worm
  • Trojan Horse
  • Spyware
  • Phishing
  • DDoS Attack
  • Software that is specifically designed to disrupt, damage, or gain unauthorized access to a computer system
  • A computer program that can copy itself and infect a computer without permission or knowledge of the user
  • A type of malware whose primary function is to self-replicate and infect other computers while remaining active on infected systems
  • A type of malware that downloads onto a computer disguised as a legitimate program
  • Malicious software that enters a user's computer, gathers data from the device and user, and sends it to third parties without their consent
  • A technique for attempting to acquite sensitive data, such as bank account numbers, through a fraudulent solicitation in email or on a web site, in which the perpetrator masquerades as a legitimate business or reputable person
  • Distrubted Denial-of-Service (DDoS) Attack is a cybercrime in which the attacker floods a server with internet traffic to prevent users from accessing connected online services and sites
Protection
  • Firewall
  • Spam Filter
  • Proxy Server
  • Antivirus Software
  • Cryptography
  • Public Key Encryption
  • A network security device that prvents unauthorized access to a network
  • A program used to detect unsolicited, unwanted and virus-infected emails and prevent those messages form getting to a user's inbox
  • A system or router that provides a gateway between users and the internet
  • A program designed to detect and remove viruses and other kinds malicious software form your computer or laptop
  • The collection of secure information and communication techniques employing mathematical concepts and algorithms used to disguise the content of messages
  • A method of encrypting or signing data with two different keys and making one of the keys, the public key, available for anyone to use
